## Getting Started

To get started, follow the installation instructions below

### Prerequisites

Must have node and npm installed to run


### Installation

1. Clone the repo

   `git clone https://github.com/404pandas/e-commerce-backend.git`

2. Run npm

   `npm i`

3. Run seeds

   `npm run seed`

4. Run server

    `npm start`
